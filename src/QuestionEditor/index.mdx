## API

|参数|说明|类型|默认值|
|:--|:--|:--|:--|
|question|传入的试题json|Object|{}|
|changeValue|修改了试题内容后传出来的值|Function|args[0]:传出来的当前框的json;args[1]:传出来的数据在json里面的定位；args[3]:综合题小题定位；args[4]:试题类型|
