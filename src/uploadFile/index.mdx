## API

|参数|说明|类型|默认值|
|:--|:--|:--|:--|
|url|上传的url地址|string(必填)|https://gateway.test.eastedu.ltd|
|token|token|string(必填)|''|
|maxSize|上传文件的最大M数|string?||
|fileNameLength|上传的文件名的限制长度|number?||
|showLists|是否显示默认文件列表|Arrey(必填)||
|multiple|接受的文件类型|string(必填)|"*"|
|fileList|已经上传的文件列表|Arrey(必填)||
|accept|接受上传的文件类型, 详见input accept Attribute|string(必填)||
|error|传出来的错误|Function(必填)||
|onChangeFileList|上传之后的数据变化|Function(必填)||
